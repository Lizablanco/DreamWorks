Top las Mejores 12 películas de DreamWorks🌙

🎥 Bienvenido al Reino de las Películas

Este proyecto es una experiencia interactiva donde cada película es una historia, cada usuario un explorador, y cada opinión una chispa de magia.

Únete a nosotros para descubrir, descargar y debatir sobre las mejores películas de DreamWorks. ¡Tu aventura cinematográfica comienza aquí!

📁 Gestión de Películas

CRUD completo para agregar, editar y eliminar películas con géneros, curiosidades y archivos descargables.

🗣️ Opiniones de Usuarios

Los visitantes pueden dejar una única opinión por película, creando una comunidad crítica y apasionada.

⬇️ Descargas Controladas

Cada usuario puede descargar una película una sola vez, con registro automático y control de acceso.

🔐 Autenticación y Seguridad

Sistema de login y registro personalizado con formularios flotantes y validaciones mágicas.

🛠️ Tecnologías Utilizadas

  • Django (Modelo-Vista-Template)
  • HTML5, CSS3 y Bootstrap
  • JavaScript para interacción dinámica
  • Autenticación con `login_required` y control por usuario
